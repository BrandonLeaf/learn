## 1. 版本种类 

* 常见JDK有两种：
    * openJDK 
    * oracleJDK（sunJDK）

## 2. openJDK

* OpenJDK是开放源码的java开发工具
* 以GPL协议的形式发布，允许在商业上使用
* 不能使用Java商标

## 3. oracleJDK

* 基于openJDK开发，其大部分源码都相同，只有少部分源码被替换掉。
* 采用JRL协议发布，只允许个人研究使用
